# Contributing

気軽に @takanakahiko までご相談ください

## はじめての人

- Fork する
- Clone する
- `git remote add upstream git@github.com:prickathon/shop-search.git`
- `git checkout -b <作業内容がわかるブランチ名>`
- `git commit -am '<コミットメッセージ>'`
- `git push origin`
- PR を作成する

## 二回目以降の人

- `git checkout master`
- `git fetch upstream`
- `git merge upstream/master`
- `git checkout -b <作業内容がわかるブランチ名>`
- `git commit -am '<コミットメッセージ>'`
- `git push origin`
- PR を作成する
